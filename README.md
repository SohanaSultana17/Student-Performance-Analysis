# Student-Performance-Analysis
# 📊 Student Performance Analysis System
## 📌 Project Overview

The **Student Performance Analysis System** analyzes student data (marks and attendance) to generate meaningful insights. It helps in identifying top performers, understanding department-wise trends, and detecting students at academic risk.

---

## 🎯 Objectives

* 📈 Calculate average marks
* 🏆 Identify top-performing students
* 🏫 Analyze department-wise performance
* 📊 Study attendance vs marks relationship
* ⚠️ Detect at-risk students
* 📉 Provide data-driven insights

---

## 🧰 Tech Stack

* **Python** (Pandas, Matplotlib)
* **SQL (MySQL)**
* **Microsoft Excel**
